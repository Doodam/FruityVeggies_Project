못난이 과일과 채소를 전문으로 하는 온라인 쇼핑몰입니다. <br>

팔지 못해 버리게 되는 못난이들을 소중히 취급하고, 고객들에게 저렴한 가격에 제공하고자 만들게 되었습니다.
